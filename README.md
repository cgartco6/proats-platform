# ProATS - AI-Powered Career & ATS Platform

Full featured ATS Resume Builder, Cover Letter, Interview Prep, AI Tutor, Payments, and Marketing Engine.

**Owner**: Unlimited free access (set passphrase in backend).

## Setup
- Backend: `cd backend && npm install && npm run dev`
- Frontend: `cd frontend && npm install && npm start`
- Mobile: `cd mobile && npm install && npx expo start`
- Docker: `docker-compose up`

Ready for 2026 career success.
